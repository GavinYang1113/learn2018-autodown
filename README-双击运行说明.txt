清华网络学堂课程下载工具 - 双击运行说明

一、使用前准备

1. 请先安装 Chrome 浏览器。
2. Windows 用户下载 learn-windows 压缩包。
3. macOS 用户下载 learn-macos 压缩包。
4. 解压后请把整个文件夹放到英文路径或普通中文路径下，不要只拿出其中某一个文件。

二、Windows 使用方法

1. 双击 run.bat。
2. 程序会自动打开 Chrome。
3. 在浏览器中登录网络学堂，完成双因素认证。
4. 登录后不要关闭浏览器窗口，等待下载完成。

三、macOS 使用方法

1. 双击 run.command。
2. 如果系统提示无法打开，可以右键 run.command，选择“打开”。
3. 程序会自动打开 Chrome。
4. 在浏览器中登录网络学堂，完成双因素认证。
5. 登录后不要关闭浏览器窗口，等待下载完成。

四、常见问题

1. 浏览器会话失效：
   删除同目录下的 session.json 后重新运行。

2. macOS 提示文件来自未知开发者：
   右键 run.command 选择“打开”，或在“系统设置 - 隐私与安全性”中允许打开。

3. 下载速度慢：
   run.bat 和 run.command 默认启用多进程下载。也可以关闭 VPN 或代理后重试。

4. 想指定下载目录或课程：
   请联系工具维护者调整 run.bat 或 run.command 里的启动参数。
